# 项目简报

## 项目名称

北京纬理律师事务所网站设计与品牌升级

## 项目目标

为北京纬理律师事务所建立一个更适合管理层战略、客户转化和律师吸引的网站方向。

项目不以“做一个漂亮官网”为目标，而以“建立信任前置的业务基础设施”为目标。

## 核心判断

1. 当前律所网站如果只是电子名片，无法支撑客户决策。
2. 纬理目前不宜硬做“大所权威感”，更适合突出“关键问题解决型律所”。
3. 互联网、平台、新型风险、复杂争议是差异化重点。
4. 律师 IP 与内容能力应成为网站战略的一部分，但不能变成泛内容门户。
5. 招聘模块不应只是“加入我们”，而应体现“与复杂问题同行”的筛选机制。

## 首页核心文案

主标题：

> 我们处理的，不只是案件，而是这个时代不断出现的新型问题

副标题：

> 以司法经验理解风险，以业务视角提供可执行的法律解决路径。

四卡片：

1. 互联网平台争议与复杂案件  
   面对平台责任、网络侵权、知识产权与批量争议，我们帮助企业建立可执行的应对路径。

2. 企业经营与新型风险判断  
   当业务模式、监管边界和刑事风险交织，我们以司法经验帮助企业提前判断风险。

3. 个人与家族关键事务  
   面对资产、关系与身份变化，我们提供克制、私密、可执行的法律安排。

4. 加入纬理：与复杂问题同行  
   如果你希望处理更具挑战的新型案件，并在专业表达中建立个人影响力，纬理提供共同成长的平台。

## 当前决策建议

建议以当前首页 UI 为第一版方向继续深化：

- 不做内容门户
- 不堆砌机构权威
- 以关键问题分流作为首页核心
- 用服务方案优势支撑信任
- 用律师 IP 模块承接内容与人才战略
