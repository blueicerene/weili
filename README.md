# 北京纬理律师事务所网站设计与品牌升级项目

## 项目定位

本项目围绕北京纬理律师事务所的网站设计与品牌升级展开，目标不是制作普通律所官网，而是建立一套面向客户决策、内容沉淀和律师吸引的业务基础设施。

核心定位：

> 我们处理的，不只是案件，而是这个时代不断出现的新型问题。

当前建议方向：

- 以“互联网时代关键问题”为主轴。
- 首页不硬做大所权威，而是通过客户问题场景建立专业感。
- 重点突出互联网平台争议、企业新型风险、个人与家族关键事务，以及律师发展与 IP 打造能力。

## 核心策略

### 1. 网站角色

网站应从“展示页”升级为：

- 信任系统：让客户快速判断律所是否可靠。
- 转化系统：让客户知道如何安全开始咨询。
- 内容系统：把复杂案件经验沉淀为专业影响力。
- 人才系统：吸引愿意处理复杂问题、有表达能力的律师。

### 2. 首页核心结构

当前首页原型采用以下结构：

1. 顶部导航与咨询入口
2. Hero：新型问题品牌主张
3. 四类问题分流卡片
   - 互联网平台争议与复杂案件
   - 企业经营与新型风险判断
   - 个人与家族关键事务
   - 加入纬理：与复杂问题同行
4. 为什么选择纬理
5. 法律服务方案优势
6. 代表案例与案例洞察
7. 专业内容与律师 IP 打造
8. 核心律师团队
9. 底部咨询 CTA

### 3. 服务方案优势表达

首页已将“我们的服务方式”升级为“法律服务方案优势”，突出律所响应文件中的真实能力：

- 稳定服务团队
- 专属项目联络人
- 2 小时紧急响应
- 1 个工作日常规反馈
- 书面与电话并行
- 方案动态调整
- 内控、保密、利冲排查
- 后续跟进与复盘

## 视觉方向

主色系统：

- 深海军蓝：专业、可信、稳重
- 暖金：行动入口、重点强调、法律权威感
- 米白：页面底色、留白、商务温度
- 暗红/砖红：少量用于内容或标签强调

设计原则：

- 克制，不做夸张营销。
- 用结构建立信任，不靠装饰堆砌。
- 用客户问题分流，不只罗列业务领域。
- 用服务机制承接转化，不只放联系方式。
- 内容模块服务于信任与律师 IP，不做泛内容门户。
